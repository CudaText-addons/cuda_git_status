plugin for CudaText.
for active file tab, it shows information about Git repo, in statusbar in the additional cell.

- branch name, e.g. "master".
- * symbol, if this branch is not clean.
- (m,n) with 2 numbers: number of commits ahead, number of commits behind.

plugin was ported from GitStatusBar plugin for Sublime Text.
plugin has config-file with few options. to edit it, call menu item
"Options / Settings-plugins / Git Status / Config", and then restart the CudaText.
